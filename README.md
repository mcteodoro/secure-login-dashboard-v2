# Secure Login Dashboard

Sistema de login seguro com autenticação de usuários.

## Tecnologias utilizadas

- HTML
- CSS
- JavaScript
- Node.js
- Express
- MongoDB Atlas
- Render
- Vercel

## Funcionalidades

- Cadastro de usuário
- Login
- Dashboard protegido
- Integração com banco de dados
- Backend online
